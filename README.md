# Wargus TypeScript

This project is the clean browser-native port of Wargus/Warcraft II gameplay.
It is not a WebAssembly wrapper around the desktop Stratagus binary.

## Current Slice

- TypeScript + PixiJS + Vite browser app.
- The default browser entry now opens a fixed playable demo on `maps/ladder/Garden of war BNE.pud.smp.gz`, staged as a readable human-vs-orc 1v1 on real Wargus terrain with fog lifted, a source-style briefing, a selected worker start, clear base/resource context, gather/train/raid/assault objectives, a starting Ballista for the final building kill, and no title-screen gate.
